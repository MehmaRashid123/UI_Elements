# UI Element

This project is a single-page web application that provides user authentication using Supabase.

## Features

- User login
- User signup
- Password reset

## Setup

1.  **Configure Supabase:**
    - Open the `config.js` file.
    - Replace the placeholder values for `SUPABASE_URL` and `SUPABASE_ANON_KEY` with your actual Supabase credentials.

2.  **Run the application:**
    - Open the `index.html` file in your web browser.

## How it works

The `config.js` file contains the Supabase credentials. The `script.js` file imports these credentials to initialize the Supabase client.
